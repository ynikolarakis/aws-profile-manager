Features

Everything you need to manage AWS at scale

Profile management, terminal, AI, cost analysis — built into one cohesive tool.

Profile Management

SSO, credentials, and role profiles organized into categories with color coding and favorites.

Built-in Terminal

Execute AWS CLI commands with real-time streaming output. Profile context set automatically.

AI Command Generation

Describe what you want in natural language. AI generates the AWS CLI command with streaming output.

Cost Analysis

Per-profile cost badges and detailed service breakdown. Spot spending anomalies instantly.

SSO Discovery

Auto-discover all accounts from AWS Organizations. One click to import and configure profiles.

Keyboard-First

Command palette with Ctrl+K, shortcuts for every action. Navigate and operate without touching the mouse.

Terminal

A terminal that knows your profile

Run AWS CLI commands directly in the app. Your active profile's credentials are injected automatically — no more exporting environment variables.

  • Bulk run commands across multiple profiles simultaneously
  • Command history persisted per profile
  • Real-time streaming output with ANSI color support
  • Identity verification on profile switch
AI Integration

Natural language to AWS CLI

Describe what you want in plain English. The AI generates the exact AWS CLI command, ready to execute with one click.

  • 6 AI providers: Bedrock, Anthropic, OpenAI, Google, OpenRouter, Ollama
  • Streaming responses with real-time token output
  • Context-aware — knows your active profile and region
  • One-click execution of generated commands
AI > list all VPCs in Frankfurt
Multi-Account

Multi Profiles. One sidebar.

Organize dozens of AWS accounts into collapsible categories. Color-code teams, search instantly, switch in one click.

  • Categories with custom colors for team organization
  • SSO at scale — discover and import from Organizations
  • Real-time search across all profiles
  • Favorites for quick access to critical accounts
Command Palette

Everything at your fingertips

  • Search across profiles, actions, and commands
  • Switch profiles instantly with keyboard navigation
  • Fuzzy matching for quick access
