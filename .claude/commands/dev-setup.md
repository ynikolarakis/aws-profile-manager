Set up the complete development environment for AWS Profile Manager:

1. Check Python 3.12+ and Node 20+ are installed
2. Create Python virtual environment and install: fastapi, uvicorn, boto3, pytest, ruff, httpx, pytest-asyncio
3. Initialize Vite + React + TypeScript project in /frontend
4. Install: tailwindcss, @shadcn/ui, zustand, @tanstack/react-query, xterm, framer-motion
5. Create Makefile with: dev, build, test, lint targets
6. Initialize git repository with .gitignore
7. Verify all MCP servers are connected
8. Report status of each step (pass/fail)

Important:
- Do NOT overwrite existing files without asking
- Use the project's CLAUDE.md for architecture reference
- Primary region is eu-central-1
- Python venv should be in .venv/ directory
